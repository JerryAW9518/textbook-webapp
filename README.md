# 國小學習小幫手 - 網頁應用程式

國小習題解答查詢平板網頁應用，採用莫蘭迪色系設計。

## ✨ 功能特色

- 📱 **平板優化** - 完美支援 iPad 和各種平板裝置
- 🎨 **莫蘭迪色系** - 柔和舒適的視覺體驗
- 📚 **完整涵蓋** - 1-6年級國文、數學全年級
- 🏫 **三大版本** - 支援翰林、康軒、南一教材
- ⚡ **快速查詢** - 即時載入解答內容

## 🚀 快速開始

### 安裝相依套件

```bash
npm install
```

### 啟動開發伺服器

```bash
npm start
```

應用程式將在 [http://localhost:3000](http://localhost:3000) 開啟

### 建置生產版本

```bash
npm run build
```

## 🎨 莫蘭迪色系配色

```css
--morandi-sage: #9CAF88;        /* 鼠尾草綠 - 主要色 */
--morandi-dusty-blue: #A4B8C4;  /* 霧藍 - 數學科 */
--morandi-rose: #D4B5B0;        /* 玫瑰粉 - 國文科 */
--morandi-beige: #D9CFC1;       /* 米色 - 背景 */
--morandi-cream: #F5F1E8;       /* 奶油色 - 主背景 */
```

## 📱 響應式設計

- **手機** (< 768px): 單欄佈局
- **平板** (768px - 1024px): 雙欄/三欄佈局
- **大螢幕** (> 1024px): 完整網格佈局
